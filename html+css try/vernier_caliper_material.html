Help & Instructions

How to Use

Customize main scale divisions, vernier scale divisions, and zero error using the sliders in the sidebar.

Creating Problems

Click "Create Problem" to generate a random measurement scenario. Drag the created object near the internal jaws, external jaws, or depth blade to snap it into position.

Controls

  • Mouse Wheel: Rotate vernier scale
  • Arrow Keys: Fine vernier control
  • Drag: Move vernier or rotate entire caliper
  • Q/E Keys: Fine rotation control
  • R Key: Reset rotation
  • Page Up/Down: Zoom in/out

Measurement

Position the object and read the measurement from the main scale and vernier scale. The total reading is calculated automatically.
